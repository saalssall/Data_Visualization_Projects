# Data Visualization Projects

A collection of data visualization projects built with Python and Jupyter Notebooks, exploring real-world datasets across health, food, and social topics.

## Projects

| # | Title | Dataset |
|---|-------|---------|
| 1 | Gun Deaths in the US (2012–2014) — Exploratory Analysis | Death data by gender, race, and cause |
| 2 | Gun Deaths in the US (2012–2014) — Visualizations | Death data with Matplotlib bar charts |
| 3 | Food Centre Orders Analysis | Food orders by category, cuisine, and pricing |
| 4 | Healthcare Data Analysis | Stroke and heart disease by gender, smoking status, and BMI |

## Tech Stack

- Python 3
- Jupyter Notebook
- Pandas · NumPy · Matplotlib · Seaborn

## Getting Started

```bash
git clone https://github.com/saalssall/Data_Visualization_Projects.git
cd Data_Visualization_Projects
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook
```
